"""
crx-expert 서브에이전트를 사용하려면?
"""

● crx-expert 서브에이전트는 제가 (메인 Claude Code) 복잡한 Chrome Extension 작업을 처리할 때 자동으로 호출합니다. 사용자가 직접    호출하는 방식이 아닙니다.

동작 방식

자동 호출 — Chrome Extension 관련 작업을 요청하면 제가 판단해서 crx-expert 에이전트에게 위임합니다.

예시:
- "Array mode 추출 기능 구현해줘" → crx-expert 호출
- "manifest.json 설정 수정해줘" → crx-expert 호출
- "content script injection 오류 디버깅" → crx-expert 호출
- "npm run build 오류 해결" → crx-expert 호출

직접 작업 요청만 하면 됩니다

"XPath 추출 로직 개선해줘"
"popup UI에 버튼 추가해줘"
"background service worker에서 메시지 처리 구현"

이런 요청을 받으면 제가 crx-expert 에이전트를 적절히 활용하여 처리합니다. 사용자는 /agents 명령으로 에이전트를 생성/편집하는 것만 할 수 있고, 에이전트 호출은 저(Claude Code)가 담당합니다.
